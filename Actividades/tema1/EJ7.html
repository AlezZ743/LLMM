Ejercicio 7 - Fondo y color del texto

Esta página tiene un fondo azul claro y el texto en color azul oscuro.

Puedes cambiar el color de fondo modificando el atributo bgcolor del elemento <body>. Por ejemplo: bgcolor="yellow", bgcolor="pink", bgcolor="black", etc.

También puedes cambiar el color del texto usando el atributo text, por ejemplo: text="white" o text="green".
